# DouyinTV - 抖音电视版应用

这是一个专为Android TV设计的抖音客户端应用,通过WebView嵌入抖音网页,实现在智能电视上浏览和播放抖音短视频。

## ✨ 功能特性

- ✅ **完整嵌入抖音网页** - 嵌入抖音官方网页,功能完整
- ✅ **电视遥控器操作支持** - 完整适配电视遥控器,操作流畅
- ✅ **自动播放下一个** - 视频播放完毕自动跳转到下一个视频
- ✅ **Cookie持久化** - 登录状态自动保存,无需重复登录
- ✅ **大屏优化显示** - 针对电视大屏优化UI和交互
- ✅ **视频流畅播放** - 硬件加速,播放流畅不卡顿
- ✅ **启动画面** - 精美的启动画面提升用户体验

## 🚀 快速开始

### 环境要求
- JDK 17+
- Android SDK
- Gradle 8.0+ (已包含Wrapper)

### 编译APK

**Windows用户** - 双击运行:
```
build.bat
```

**手动编译**:
```powershell
# Release版本(推荐)
.\gradlew assembleRelease

# Debug版本
.\gradlew assembleDebug
```

### APK位置
```
app\build\outputs\apk\release\app-release.apk
```

## 📺 安装到电视

1. 将APK复制到U盘
2. U盘插入电视
3. 使用文件管理器安装
4. 允许"安装未知来源应用"
5. 启动应用开始使用

## 🎮 遥控器操作

| 按键 | 功能 |
|------|------|
| **上/下键** | 切换视频 |
| **左/右键** | 水平滚动 |
| **确认键** | 播放/暂停 |
| **返回键** | 返回上一页 |
| **菜单键** | 显示菜单 |

## 🛠️ 技术栈

- **语言**: Kotlin 1.9.0
- **框架**: Android WebView + AndroidX
- **TV支持**: Leanback库
- **最低版本**: Android 5.0 (API 21)
- **目标版本**: Android 14 (API 34)

## ⚠️ 注意事项

- 首次使用需要登录抖音账号
- 建议使用WiFi网络获得最佳体验
- 部分功能可能因抖音网页更新而受影响
- 仅供学习交流使用,不得用于商业用途
